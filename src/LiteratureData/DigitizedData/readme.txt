
1) Screenshot all individual axes to be digitized and save them in the same folder (see Template folder)
2) Open https://apps.automeris.io/wpd/
3) Load all images for the same paper 
4) For each image (see video example if needed):
	a) calibrate the axes 
	b) digitize the plot
	c) save data (click view data + "download.CSV")
5) Copy all the CSV files to the folder "CSVfiles"
6) Run "DigitizedData.mat" to convert CSV files to mat format
